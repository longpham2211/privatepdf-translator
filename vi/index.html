Cách Sử Dụng

Chỉ 3 bước đơn giản để xử lý tài liệu PDF của bạn

1

Chọn công cụ

Duyệt qua các công cụ ở trên và chọn công cụ phù hợp với nhu cầu của bạn — ghép, nén, chuyển đổi, hoặc dịch tài liệu.

2

Tải file lên

Kéo thả file PDF vào công cụ. Các công cụ xử lý trên trình duyệt đảm bảo file không rời khỏi thiết bị của bạn.

3

Tải về kết quả

Khi xử lý hoàn tất, tải file về ngay lập tức. Không cần tạo tài khoản cho các công cụ cơ bản.

Các gói & giá

Chọn gói phù hợp với nhu cầu sử dụng của bạn. Hiển thị tiền tệ tự động theo ngôn ngữ.

Câu Hỏi Thường Gặp

Làm thế nào để ghép file PDF?
Vào công cụ Hợp nhất PDF, kéo thả các file PDF, sắp xếp thứ tự, và bấm Ghép. File kết quả sẽ tự động tải về. Công cụ này xử lý hoàn toàn trên trình duyệt của bạn.
File của tôi có an toàn không?
Có! Các công cụ xử lý trên trình duyệt (ghép, tách, nén, xoay, watermark, ký tên) không bao giờ gửi file ra ngoài. Các công cụ server sử dụng mã hóa và xóa file ngay sau khi xử lý xong.
PrivatePDF có thực sự miễn phí không?
Đúng vậy. Các công cụ cơ bản hoàn toàn miễn phí không giới hạn. Tính năng cao cấp như Dịch PDF, Chat với PDF, OCR và hỗ trợ file lớn (đến 500MB) có trong gói Pro.
Dung lượng file tối đa là bao nhiêu?
Người dùng miễn phí có thể xử lý file đến 50MB. Người dùng Premium xử lý file đến 500MB. Tốc độ xử lý trên trình duyệt phụ thuộc vào cấu hình thiết bị.
Tôi có cần tạo tài khoản không?
Không cần tài khoản cho các công cụ cơ bản. Bạn có thể ghép, tách, nén file ngay lập tức. Tài khoản chỉ cần cho tính năng cao cấp và lưu cài đặt cá nhân.
Có thể dùng trên điện thoại không?
Có. PrivatePDF tương thích hoàn toàn với mọi thiết bị có trình duyệt hiện đại — Android, iPhone, iPad. Không cần tải app.
Hỗ trợ những định dạng file nào?
PrivatePDF hỗ trợ PDF, Word (.docx), Excel (.xlsx), PowerPoint (.pptx), hình ảnh (JPG, PNG, WEBP), và văn bản thuần (TXT). Mọi chuyển đổi đều giữ nguyên bố cục gốc một cách tốt nhất.
