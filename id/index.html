Cara Menggunakan

Hanya 3 langkah sederhana untuk memproses dokumen PDF Anda

1

Pilih alat

Telusuri alat di atas dan pilih salah satu yang sesuai dengan kebutuhan Anda — gabungkan, kompres, konversi, atau terjemahkan.

2

Unggah file Anda

Seret dan lepas PDF Anda ke dalam alat. Alat berbasis browser menyimpan file Anda di perangkat Anda.

3

Hasil unduhan

Jika proses sudah selesai, segera download filenya. Tidak diperlukan akun untuk alat dasar.

Pertanyaan yang Sering Diajukan

Bagaimana cara menggabungkan file PDF?
Buka Gabungkan PDF, seret dan lepas file Anda, susun urutannya, lalu klik Gabung. File keluaran akan diunduh secara otomatis. Alat ini sepenuhnya berjalan di browser Anda.
Apakah file saya aman?
Ya. Alat berbasis browser (menggabungkan, membagi, mengompres, memutar, memberi tanda air, menandatangani) tidak pernah mengirim file keluar dari perangkat Anda. Alat server menggunakan enkripsi dan menghapus file secara otomatis setelah pemrosesan.
Apakah PrivatePDF benar-benar gratis?
Ya. Alat inti sepenuhnya gratis tanpa batasan harian. Fitur premium seperti Terjemahan PDF, Obrolan PDF, OCR, dan dukungan file besar (hingga 500MB) tersedia dengan Pro.
Berapa ukuran file maksimum?
Pengguna gratis dapat memproses file hingga 50MB. Pengguna premium dapat memproses hingga 500MB. Kecepatan pemrosesan browser bergantung pada perangkat Anda.
Apakah saya memerlukan akun?
Tidak diperlukan akun untuk alat dasar. Anda hanya memerlukan akun untuk fitur premium dan pengaturan tersimpan.
Bisakah saya menggunakannya di ponsel?
Ya. PrivatePDF berfungsi di browser modern di Android, iPhone, dan iPad. Tidak diperlukan instalasi aplikasi.
Format file apa yang didukung?
PrivatePDF mendukung PDF, Word (.docx), Excel (.xlsx), PowerPoint (.pptx), gambar (JPG, PNG, WEBP), dan teks biasa (TXT).
